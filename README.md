# 🐱 Random Cat Fact

A simple Android app that fetches random cat facts from an API and allows users to save their favorite facts locally.

## ✨ Features

* Fetch random cat facts from a remote API
* Save and remove favorite facts
* Toggle favorite state using a heart button
* View saved favorite facts
* Loading and error state handling

## 🛠 Tech Stack

* **Kotlin**
* **Jetpack Compose**
* **MVVM Architecture**
* **StateFlow**
* **Hilt** (Dependency Injection)
* **Retrofit** (Networking)
* **Room** (Local Database)
* **Coroutines**
